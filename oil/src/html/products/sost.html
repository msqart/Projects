Смазочно-охлаждающие технологические средства для обработки металлов давлением

Пластические свойства металлов позволяют предавать им нужную форму после воздействия на них давления. При этом получают как заготовки, так и уже готовые изделия. Существуют разные технологии обработки металлов давлением, как горячих, так и холодных заготовок и, соответственно, разные смазочно-охлаждающие технологические средства для каждой операции:

Прокатка

Процесс пластического деформирования тел между вращающимися приводными валками прокатного стана, которые обжимают заготовку для придания ей нужной формы и размера. В зависимости от выпускаемых изделий бывают: трубопрокатные, листопрокатные, проволочные и другие виды прокатных станов.

Прессование

Способ обработки металла, в процессе которого он выдавливается прессом из полости матрицы через ее отверстие. Прессованием получают полые и сплошные профили из титановых сплавов, алюминия, меди и магния – детали для автомобилей, самолетов, лопатки, подвески, трубы.

Прессование может быть горячим и холодным. Тугоплавкие металлы и сплавы (содержащие никель, титан и др.) прессуются после предварительного нагрева заготовки и инструмента. Обработка пластичных материалов (олово, медь, чистый алюминий) производится в холодном состоянии. Между инструментом и обрабатываемой деталью возникает экстремальное удельное давление. Для решения этой задачи необходимы разделительные среды, способные образовывать плёнку, устойчивую к воздействию высокого давления и предотвращать износ инструмента:

  • Не смешиваемые с водой масла для холодной формовки, с высокоэффективным пакетом противозадирных и противоизносных присадок.

Ковка

Ковка применяется в единичном и мелкосерийном производстве и относится к высокотемпературным операциям. Заготовка предварительно нагревается до ковочной температуры, зависящей от конкретной марки металла, чтобы металл был более пластичным. Для получения поковки разогретую заготовку помещают между ударными частями (бойками) молота.

Применение разделительных средств в процессе ковки или горячего прессования способствует улучшению качества поверхности, обрабатываемых деталей и увеличению срока службы инструмента. В качестве разделительных сред используют:

  • Смешиваемые с водой СОТС для нанесения на инструмент, в виде дисперсии твёрдых смазочных материалов;
  • СОТС в виде дисперсии или суспензий твёрдых смазочных материалов в масле.
  • Порошки твёрдых смазочных материалов.

Объёмная штамповка

Метод обработки металла, при котором придание заготовке заданной формы осуществляется с помощью штампов и определяется их конфигурациями. Штамповка может производиться как с предварительным нагревом, так и без него. При этом течение металла ограничивается полостями его частей. В качестве разделительных сред используют:

  • Технологическое средство на базе воды в виде суспензии с воскообразными компонентами;
  • Суспензии со светлыми твёрдыми смазочными материалами для горячей штамповки сложных деталей;
  • Пасты с высоким содержанием твердых смазочных материалов.

Листовая штамповка

Листовая штамповка заключается в изготовлении плоских и пространственных деталей из листа, ленты, полосы. В качестве инструмента для деформации заготовки используются штампы, основными деталями которых являются матрицы и пуансоны. В зависимости от требований к детали и сложности операции применяют следующие среды:

  • Испаряющиеся жидкости для штамповки;
  • Не смешиваемые с водой масла для штамповки и глубокой вытяжки;
  • Эмульгируемые среды для штамповки и глубокой вытяжки;
  • Пасты для глубокой вытяжки.

Волочение

Способ обработки металлической заготовки круглого (или фасонного) профиля, которая протягивается через фильеру (волоку). В результате происходит увеличение длины заготовки и уменьшение ее поперечного сечения. Примером может служить изготовление проволоки, когда заготовка большого диаметра (катанка) протягивается через ряд фильер, на выходе из которых получается проволока гораздо меньшего диаметра. Для разных типов волочения используются:

  • Мыльные порошки для сухого волочения;
  • Эмульсии или растворы для мокрого волочения;
  • Пасты, смешиваемые с водой, для мокрого волочения;
  • Масла для мокрого волочения.
